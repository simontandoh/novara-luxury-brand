# Novara Template

Reusable, generic Next.js App Router starter for Novara projects.

## What is included

- Next.js App Router with TypeScript
- Minimal dependency set
- Generic placeholder homepage
- Preserved top-level structure for `app/`, `public/`, `components/`, `lib/`, `styles/`, and `docs/`

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Project structure

- `app/`: App Router entrypoints, layout, pages, and global styles
- `components/`: Reusable UI components
- `lib/`: Shared config and utility code
- `styles/`: Shared design tokens and style primitives
- `public/`: Static assets
- `docs/`: Internal notes for maintainers and future setup

## Notes

- The homepage is intentionally a placeholder, not a finished site.
- Replace placeholder content before using the template for a real project.
- Keep additional dependencies justified and minimal.
